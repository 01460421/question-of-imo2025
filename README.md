# IMO 2025 Problem 5 分析系統

**International Mathematical Olympiad 2025 Problem 5 — Interactive Mathematical Analysis System**

## 問題描述

Alice 和 Bazza 輪流選擇非負實數。在第 *n* 回合，當前玩家選擇一個非負實數 *xₙ*，使得目前為止所選的所有數字滿足以下兩個約束條件：

1. **線性約束**：∑xᵢ ≤ λn
2. **二次約束**：∑xᵢ² ≤ n

若某玩家無法選擇合法的 *xₙ* ≥ 0，則該玩家落敗。

## 臨界值定理

臨界值為 **λ* = 1/√2 ≈ 0.707107**

- 若 λ > λ*，則 Alice 有必勝策略
- 若 λ < λ*，則 Bazza 有必勝策略
- 若 λ = λ*，遊戲趨於平衡

## 功能

### 模擬分析
- 自訂 λ 值與最大回合數
- 5 種玩家策略：保守型、激進型、平衡型、最優型、適應型
- 4 種約束類型：標準、立方、四次方、加權
- 圖表可視化與詳細統計
- CSV/JSON 匯出

### 批次掃描
- **自訂參數**：起始 λ、結束 λ、步長、回合數
- 系統性驗證臨界值理論
- 勝負統計與散點圖
- **冪次函數擬合**：自動擬合 n = a × |λ − λ*|^b + d

### 臨界值分析
- **自訂參數**：掃描範圍、步長、回合數
- 高精度相變圖
- **冪次函數擬合**：高精度估計冪次指數
- R² 與 RMSE 統計

### 策略對比
- **自訂參數**：λ 值、回合數
- 25 種策略組合測試
- 勝負矩陣顯示

### 理論說明
- 完整數學推導與證明
